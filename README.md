# CephasGM AI

CephasGM AI is an African-inspired AI platform designed to evolve into a full artificial intelligence assistant similar to ChatGPT, Gemini, and DeepSeek.

## Features

- Progressive Web App (PWA)
- React frontend
- Optional Firebase authentication
- Offline support
- Installable mobile app

## Tech Stack

React  
Firebase  
JavaScript  
Service Workers  
PWA Manifest  

## Future Phases

Phase 2 will include:

- AI chat backend
- Image generation
- Document analysis
- Video generation
- File uploads
- AI voice interaction
- AI coding assistant
